# find

- 주어진 범위 내에서 val에 해당하는 값의 Iterator를 리턴합니다.

```cpp
template <typename It, typename Val>
It my_find(It begin, It end, Val val){
    for (It start = begin; start < end; start++){
        if(*start == val)
            return start;
    }
    return end;
}
```
